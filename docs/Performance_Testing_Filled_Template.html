Step 4: Observations & Analysis

Key Findings:

The Smart Lender prediction system demonstrated excellent performance. The Random Forest model produces predictions with an average latency of 23.83 ms, yielding a throughput of approximately 41.96 predictions per second. All 367 test cases were processed without a single error.

Bottlenecks Identified:

1. Flask single-threaded dev server cannot handle concurrent requests efficiently.
2. One-hot encoding is run per request, which is slow for bulk API calls.
3. No caching mechanism for identical inputs.

Optimization Steps Taken:

1. Model is loaded once at application startup (not per-request).
2. Feature names are pre-loaded and cached in memory.
3. Default fill values are used for missing columns to avoid slow loops.
